## 🎥 How to Use Video Capture

1. Enter the website URL in the frameup window.  
2. Select **Video** mode to record a scroll-through video.  
3. Click **Start**. frameup will open the site in the browser and scroll automatically.  
4. When finished, the recorded video saves in MP4 format.  
5. Open the video with any media player to review.

---

## 📸 How to Use Screenshot Capture

1. Enter the website URL.  
2. Choose **Screenshot** mode.  
3. Click **Start** to take a full-page screenshot.  
4. The screenshot saves as a PNG file by default.  

The screenshot captures the entire webpage, even the parts you must scroll to see.

## ⚙ Settings and Options

frameup lets you adjust some options:

- **Output folder:** Choose where to save files.  
- **Image format:** PNG or JPEG for screenshots.  
- **Video quality:** Low, medium, or high resolution.  
- **Scroll speed:** Set how fast the page scrolls during video capture.  
- **Browser choice:** Use default or specify Chrome or Edge.  

Changes apply next time you run the capture.

## 🛠 Troubleshooting

- If the app does not open, check you have the latest Windows updates.  
- If capture fails, ensure the website URL is correctly typed and accessible.  
- For video capture issues, confirm your browser is up to date.  
- Close other applications that may use heavy Internet or CPU resources to improve performance.  
- Restart frameup or your computer before trying again if errors persist.
