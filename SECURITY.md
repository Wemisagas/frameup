# Security Policy

## Supported Versions

Only the latest version of frameup is actively maintained.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, open a [GitHub Security Advisory](https://github.com/zjebinsky/frameup/security/advisories/new) or contact the maintainer directly.

Include as much detail as possible:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect a response within 48 hours. If the vulnerability is confirmed, a fix will be prioritised and a patched release will be published as soon as possible.
